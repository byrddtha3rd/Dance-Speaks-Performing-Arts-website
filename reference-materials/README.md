# Reference Materials

This folder collects the information and assets needed to plan, build, and
maintain the Dance Speaks Performing Arts Academy website.

## Folders

- `brand/` for logos, color notes, type notes, and visual direction.
- `photos/` for approved image assets grouped by use case.
- `content/` for written website copy and program information.
- `existing-site/` for notes from the current or previous website.

Add real assets and source notes here before using them in the public website.
