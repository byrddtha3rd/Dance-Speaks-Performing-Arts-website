# Reference Materials

This folder collects the information and assets needed to plan and build the Dance Speaks Performing Arts website.

## Folders

- `brand/` for logos, color notes, type notes, and visual direction.
- `photos/` for image assets grouped by use case.
- `content/` for written website copy and program information.
- `existing-site/` for notes from the current or previous website.

Add real assets and source notes here before the website build begins.
