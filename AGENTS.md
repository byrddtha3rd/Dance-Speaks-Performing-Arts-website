# Codex Working Notes

## Project

Dance Speaks Performing Arts website.

## Current Phase

The repository now contains the Phase 1 public website and organized reference
materials. Future work should continue to treat `reference-materials/` as the
source of truth.

## Guidelines

- Keep reference content in `reference-materials/`.
- Do not add generated placeholder logos or photos.
- Preserve source files and public assets separately.
- Keep content drafts readable and easy for non-technical collaborators to update.
- Do not invent studio facts, staff credentials, class times, prices,
  testimonials, contact information, or policies.
- Phase 1 is public-only; do not add authentication, payments, billing,
  enrollment processing, attendance, messaging, or studio administration until
  explicitly requested.
